# carlosfpontes.github.io

[Link para o Site](https://carlosfpontes.github.io)

## Descrição do Projeto

Este diretório contém um website que apresenta um instrumento para aplicação de experimento. Trata-se de um questionário que avalia 9 estímulos diferentes, apresentados em ordem aleatória a cada acesso. Os questionários enviam as respostas para formulários do Google Forms.
